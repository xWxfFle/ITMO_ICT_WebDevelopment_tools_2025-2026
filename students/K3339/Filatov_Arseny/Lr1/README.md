# Лабораторная работа 1

## Тема

Сервис учёта личных финансов: кошельки, категории, транзакции (доходы и расходы), теги

## Этапы

| Этап | Содержание |
|------|------------|
| practice_1_1 | FastAPI, Pydantic, in-memory хранилище, CRUD |
| practice_1_2 | PostgreSQL, SQLModel, связи 1:N и M:N, роутеры по модулям |
| practice_1_3 | Alembic, переменные окружения, JWT, регистрация и смена пароля |

## Документация API

Интерактивная схема: `http://127.0.0.1:8000/docs` после запуска приложения.
