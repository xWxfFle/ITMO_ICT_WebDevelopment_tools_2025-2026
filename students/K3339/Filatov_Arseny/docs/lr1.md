# Лабораторная работа 1 — обзор

## Тема

**Сервис учёта личных финансов (Ledger):** кошельки, категории с лимитом расходов, транзакции (доход/расход), теги. Связь транзакций и тегов — **многие-ко-многим** через таблицу связи с полями **`linked_at`** и **`label_note`**.

## Итоговая модель данных (БД с 1.2 / 1.3)

| Сущность | Роль |
|----------|------|
| `user` | Владелец данных; с 1.3 — `password_hash` |
| `wallet` | Счёт / кошелёк |
| `category` | Статья бюджета, опционально `spending_cap` |
| `tag` | Метка для транзакций |
| `fin_transaction` | Операция и сумма |
| `transaction_tag_link` | M:N + `linked_at`, `label_note` |
